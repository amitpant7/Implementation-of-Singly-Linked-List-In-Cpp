# Implementation-of-Singly-Linked-List-In-C-
A menu driven program to illustrate basic operations of Singly Linked List
Operations:
        1. Insert at first
        2. Insert at last
        3. Insert at nth position
        4. Delete from first
        5. Delete from last
        6. Delete from nth position
        7. Traverse all the nodes
        8. Search any value
